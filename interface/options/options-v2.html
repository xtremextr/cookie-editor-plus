General Options

When Enabled, all the values of a cookie are visible in the interface. When Disabled, only the name and the value of the cookie are visible.
When Enabled, Cookie-Editor will be added to the DevTools tabs. Devtools needs to be restarted to take effect when turning this option off.
When Enabled, Cookie-Editor will show transitions and other various animations in the interface. This is for appearance only.
Selects the behavior of the export button. When "Ask every time" is selected, a menu asking you to chose the format will show up when you press the export button.
When Enabled, Cookie-Editor will display some small non-intrusive ads at the top of the main interface. These are used to cover the basic operating costs of Cookie-Editor. Feel free to disable them, but keep in mind that they help keep Cookie-Editor free and running smoothly!
